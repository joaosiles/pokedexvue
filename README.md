# PokedexVue

Projeto para aprendizado de VueJS

# Tecnologias

**Vue.js**

**Bulma** - CSS

**Axios** - Requisições HTTP

## Project setup  
```  
npm install  
```  
  
### Compiles and hot-reloads for development  
```  
npm run serve  
```  
  
### Compiles and minifies for production  
```  
npm run build  
```  
  
### Lints and fixes files  
```  
npm run lint  
```
